Movie Project

Context

We (me and my partner) were asked to create a simple idea for a movie. This means we were making just an introduction, not really going all out to a script or anything. The important part that we were doing was making it look good, and interesting.

Planning

We organized what each person would do towards everything of the project. This includes the storyline, images, fonts, etc.. The whole project was split into three parts: Story, HTML, and CSS

The name we decided for was 'The Hourglass Key', which was a name based off of the storyline/introduction of the 'movie'.

Content

For the story of the project, we broke it down into three sections:

  • Main screen - Main introduction
  • Sequel - Like a different side of the story
  • While not really being a part of the story, the questions screen

    • Each one of these had it's own HTML

For the HTML section of the project, we broke it down into these 6 sections:

  • Headings
  • Paragraphs
  • DIVs and Spans
  • Importing fonts
  • Inputs (submit buttons, normal buttons, etc..)
  • Images (at least two local)

For the CSS section, we brke it down into these _ sections:

  • Adjusting headings
  • Aligning everything properly
  • Making the paragraphs readable by adjusting their size, color, and font for some words (words would be in spans)

Challenges

After we were done doing the essentials of the project, we wanted to make it look good and be an overall better design. To do this, outside of our basic knowledge of CSS, we had to get creative. I went on logomakr to create a logo for our movie, and then added that image within an href to make it clickable to go back to the main page (since we had 3 different HTML pages). Also, to make the heading more interesting, I searched for a long time to find how to make it gradient, which went really well with the underline. This was the way to make a text gradient that I figured out after ~30 minutes of searching:

background: linear-gradient(to [direction], [color 1], [color 2]);

background-clip: text;

-webkite-text-color-fill: transparent;
